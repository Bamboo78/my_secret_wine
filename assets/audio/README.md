# Audio Assets

Coloca aquí tu archivo de audio para el fondo de la app.

Nombre recomendado: background.mp3
Formato: .mp3 (bitrate razonable, por ejemplo 128-192 kbps)
Duración: idealmente corto y en loop (30-120s).

Ruta final dentro del proyecto: assets/audio/background.mp3

Después de añadir el archivo, corre:

flutter pub get

y luego prueba la app en modo debug o construye la APK:

flutter build apk --release
